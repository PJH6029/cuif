# Repository purpose
- This repo collects research, meeting materials, and benchmark-design notes for CUIF: a computer-use instruction-following benchmark focused on agents that create or edit visual office artifacts such as slides, documents, and spreadsheets.
- The core research question is how to evaluate whether computer-use agents satisfy textual, visual, spatial, and mixed constraints in generated outputs, rather than only completing broad office tasks.
